## About Project

This is simple RESTful API project covered by PHPUnit tests. Also with validation rules for requests.
There we have resources Client and Project, which have one to many relationship.

"Clients" structure:
- first name
- last name
- email
- password

"Projects" structure:
- name
- description
- statuses
